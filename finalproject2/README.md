# Обучение модели с помощью классических алгоритмов (CatBoost) и подготовка фичей для рекомендаций

Кратко:
- В этой папке находятся ноутбук и код для подготовки фичей и обучения модели рекомендаций.
- API-эндпоинты сервиса сохранены из `finalproject1`, но метод рекомендаций изменён: теперь выдача основана на обученной модели (CatBoost).

Что делает `2.ipynb`:
- Сбор и предобработка данных активности (`feed`) и содержимого постов.
- Извлечение текстовых эмбеддингов tf-idf для контента постов.
- Применение PCA / KMeans для снижения размерности и получения кластерных признаков.
- Построение дополнительных фич: расстояния до центров кластеров, статистики по текстам.
- Объединение контентных фич с данными взаимодействий и обучение CatBoost-классификатора (с сохранением модели).
- Оценка качества (ROC-AUC) и сохранение артефактов (модель, фичи).

Ключевые файлы:
- `2.ipynb` — основной ноутбук подготовки фич и обучения модели.
- `app.py` — FastAPI-приложение (эндпоинты, загрузка модели при старте, `/health`).
- `requirements.txt` — зависимости для окружения.

Изменения в API по сравнению с `finalproject1`:
- Все базовые эндпоинты (`/`, `/user/{user_id}`, `/post/{post_id}`, `/user/{user_id}/feed`, `/post/{post_id}/feed`) сохранены и работают как раньше.
- Эндпоинт рекомендаций (`/post/recommendations/`) теперь использует предсказания обученной модели (CatBoost) вместо простого агрегирования лайков.
  - Входные фичи формируются из сохранённых артефактов (эмбеддинги, PCA/KMeans признаки, исторические признаки).
  - Модель загружается в `@app.on_event('startup')` и используется для ранжирования/оценки вероятности лайка.

Запуск (локально, Windows PowerShell):

```powershell
cd finalproject2
python -m venv .venv
& .venv\Scripts\Activate.ps1
pip install -r requirements.txt
# запустить API
python -m uvicorn app:app --reload --host 127.0.0.1 --port 8000
```

Проверка эндпоинтов:
- Откройте `http://127.0.0.1:8000/docs` для Swagger UI и подробной схемы.
- Пример запроса рекомендаций (curl):

```powershell
curl "http://127.0.0.1:8000/post/recommendations/?user_id=42&limit=10"
```
