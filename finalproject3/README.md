# Обучение модели с помощью глубокого обучения (DistilBERT)

Краткое описание
- Этот ноутбук подготавливает фичи для контентной рекомендации постов и обучает модель CatBoost.
- Основной поток: извлечение текстов постов → получение эмбеддингов через DistilBERT → PCA → KMeans → добавление расстояний до центров как фичи → объединение с данными активности (feed) → обучение CatBoost.

Требования
- Python 3.8+; пакетный список в `requirements.txt` в этой папке.

Как запустить (локально)
1. Создайте виртуальное окружение и активируйте его (Windows PowerShell):

```powershell
python -m venv .venv
& .venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Откройте и запустите ноутбук `3.ipynb` последовательно по ячейкам.


Файлы
- `3.ipynb` — основной ноутбук (этот репозиторий).
- `requirements.txt` — список зависимостей для окружения.
